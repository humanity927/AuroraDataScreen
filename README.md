# 极光数据大屏

`AuroraDataScreen` 是一个公开开源的数据大屏项目，用于生产实习场景下的数据可视化展示与分析。

## 项目简介

本项目面向生产实习过程中的数据展示需求，计划通过数据大屏的形式呈现生产、实习、运行状态等关键信息，帮助用户更直观地观察数据变化、理解业务流程并完成实习成果展示。

## 项目信息

- 英文名称：`AuroraDataScreen`
- 中文名称：极光数据大屏
- 项目类型：数据可视化大屏
- 开源协议：MIT
- 仓库性质：公开仓库
